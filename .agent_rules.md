# Diretrizes do Agente - InspetorAI

Este arquivo serve como configuração de memória para o assistente de IA (Antigravity).

## Regras de Comunicação e Desenvolvimento
1. **Idioma Único**: Toda e qualquer comunicação com o usuário deve ser em **Português (Brasil)**.
2. **Código**: 
   - Comentários no código devem ser em português.
   - Mensagens de log printadas no console/logger devem ser em português.
3. **Git**: 
   - Mensagens de commit devem ser escritas em português.
4. **Artefatos**: 
   - Todos os arquivos de planejamento (`implementation_plan.md`), tarefas (`task.md`) e walkthroughs (`walkthrough.md`) devem ser mantidos em português.

## Fluxo de Deploy Automatizado ("Suba o app")
Sempre que o usuário solicitar "suba o web app", "rode o app" ou similar, o procedimento obrigatório é:
1. **Sincronizar GitHub**: Commitar e dar push de todas as alterações pendentes.
2. **Validar Testes**: Executar todos os testes unitários (`pytest` ou similar).
3. **Executar Deploy**: Rodar o script de deploy para Cloud Run.
4. **Verificação Pós-Deploy**: Monitorar ativamente os logs do Cloud Run por 2 minutos em busca de exceptions.
5. **Ciclo de Correção**: Se houver exception no log, aborte a finalização, corrija o erro silenciosamente e reinicie o fluxo até que o deploy esteja limpo.
